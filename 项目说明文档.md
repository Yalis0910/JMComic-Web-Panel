# JMComic-Crawler-Python 项目详解

## 一、项目简介

**JMComic-Crawler-Python** 是一个用于爬取禁漫天堂（18comic）的 Python API 库，提供了完整的接口封装，支持网页端和移动端两种访问方式。

### 核心功能

- **漫画下载**：下载指定漫画的所有章节图片，支持自动解码还原
- **搜索功能**：支持站内搜索、标签搜索、分类搜索
- **收藏夹管理**：获取和管理个人收藏夹
- **排行榜浏览**：访问周排行、月排行等
- **GitHub Actions 集成**：可在 GitHub 上直接运行下载任务

## 三、使用方法

### 1. 命令行方式（最简单）

```bash
# 下载漫画 ID 为 123 的本子
jmcomic 123

# 同时下载多个本子
jmcomic 123 456 789

# 下载指定章节（photo）
jmcomic 123 p456

# 使用配置文件
jmcomic 123 --option="D:/option.yml"

# 查看本子详情（不下载）
jmv 350234
```

### 2. Python 代码方式

#### 2.1 基础下载

```python
import jmcomic

# 下载单个本子
jmcomic.download_album('123')

# 下载章节
jmcomic.download_photo('456')

# 同时下载多个本子
jmcomic.download_album([123, 456, 789])
```

#### 2.2 使用配置文件下载

**第一步：创建配置文件 `option.yml`**

```yaml
download:
  image:
    suffix: .png  # 转换图片格式为 PNG
  threading:
    image: 30     # 同时下载 30 张图片
    photo: 16     # 同时下载 16 个章节

dir_rule:
  base_dir: D:/jmcomic/  # 下载根目录
  rule: Bd / Atitle / Ptitle  # 文件夹命名规则
```

**第二步：运行代码**

```python
import jmcomic

# 通过配置文件创建 option
option = jmcomic.create_option_by_file('D:/option.yml')

# 使用 option 下载
jmcomic.download_album(123, option)
# 或者
option.download_album(123)
```

#### 2.3 搜索漫画

```python
from jmcomic import *

client = JmOption.default().new_jm_client()

# 站内搜索
page = client.search_site(search_query='+MANA +无修正', page=1)

# 遍历搜索结果
for album_id, title in page:
    print(f'[{album_id}]: {title}')

# 按标签搜索
page = client.search_tag('無修正', page=1)
```

#### 2.4 获取收藏夹

```python
from jmcomic import *

option = JmOption.default()
client = option.new_jm_client()
client.login('用户名', '密码')

# 遍历收藏夹
for page in client.favorite_folder_gen():
    for aid, atitle in page.iter_id_title():
        print(f'本子ID: {aid}, 标题: {atitle}')
```

#### 2.5 获取排行榜

```python
from jmcomic import *

client = JmOption.default().new_jm_client()

# 月排行
page = client.month_ranking(1)

# 周排行
page = client.week_ranking(1)

# 遍历结果
for aid, atitle in page:
    print(aid, atitle)
```

## 四、配置文件详解

### 完整配置示例

```yaml
# 日志开关
log: true

# 客户端配置
client:
  # 客户端实现：html（网页端）或 api（APP端）
  impl: html
  
  # 域名配置（可选）
  domain:
    html:
      - 18comic.vip
      - 18comic.org
    api:
      - www.jmapiproxyxxx.vip
  
  # 重试次数
  retry_times: 5
  
  # 请求配置
  postman:
    meta_data:
      # 代理配置
      proxies: system  # 使用系统代理
      # proxies: 127.0.0.1:7890  # 自定义代理
      
      # Cookies 配置（登录用）
      cookies:
        AVS: your_avs_cookie_value

# 下载配置
download:
  cache: true          # 开启缓存，避免重复下载
  image:
    decode: true       # 自动解码混淆图片
    suffix: .jpg       # 图片格式转换
  threading:
    image: 30          # 同时下载图片数
    photo: 16          # 同时下载章节数

# 文件夹规则
dir_rule:
  base_dir: D:/jmcomic/
  rule: Bd / Atitle / Ptitle
  normalize_zh: zh-cn  # 繁体转简体

# 插件配置
plugins:
  after_init:
    - plugin: login    # 登录插件
      kwargs:
        username: your_username
        password: your_password
    
    - plugin: find_update  # 只下载新章节
      kwargs:
        145504: 290266

  after_album:
    - plugin: zip      # 压缩文件插件
      kwargs:
        zip_dir: D:/jmcomic/zip/
        suffix: zip
        filename_rule: Atitle
        delete_original_file: true
    
    - plugin: img2pdf  # 转换为 PDF
      kwargs:
        pdf_dir: D:/pdf/
        filename_rule: Aname
```

## 六、项目特点

1. **绕过 Cloudflare 反爬虫**
2. **实现禁漫 APP 接口最新加解密算法**
3. **多种使用方式**：
   - GitHub Actions：网页上直接输入 ID 下载
   - 命令行：无需写代码
   - Python API：最灵活的方式
4. **双客户端支持**：
   - 网页端：效率高，但限制 IP 地区
   - APP 端：兼容性好，不限 IP
5. **多线程下载**：可细化到一图一线程
6. **可配置性强**：支持域名、代理、格式转换、路径规则等
7. **可扩展性强**：支持自定义插件和回调函数

## 八、常见使用场景

### 场景 1：批量下载收藏夹

```python
from jmcomic import *

option = create_option_by_file('option.yml')
client = option.new_jm_client()
client.login('username', 'password')

# 获取收藏夹所有本子 ID
aid_list = []
for page in client.favorite_folder_gen():
    for aid, _ in page.iter_id_title():
        aid_list.append(aid)

# 批量下载
download_album(aid_list, option)
```

### 场景 2：下载后自动压缩

```yaml
# option.yml
plugins:
  after_album:
    - plugin: zip
      kwargs:
        zip_dir: D:/jmcomic/zip/
        filename_rule: Atitle
        delete_original_file: true
```

### 场景 3：下载后转 PDF

```yaml
# option.yml
plugins:
  after_album:
    - plugin: img2pdf
      kwargs:
        pdf_dir: D:/pdf/
        filename_rule: Aname
```

### 场景 4：使用 GitHub Actions 自动下载

1. Fork 项目到自己的 GitHub
2. 进入 Actions 页面
3. 选择 "Download" workflow
4. 输入漫画 ID 运行

详细教程：[GitHub Actions 使用教程](./assets/docs/sources/tutorial/1_github_actions.md)

## 九、注意事项

1. **请勿过度爬取**：为了减轻服务器压力，不要一次性下载太多本子
2. **Python 版本**：推荐使用 Python 3.12+
3. **代理配置**：部分地区需要配置代理才能访问
4. **登录需求**：部分敏感内容需要登录才能访问
5. **Cookies 配置**：配置 cookies 时注意域名匹配
